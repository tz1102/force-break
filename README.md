# 🛡️ ForceBreak - 保护颈椎与视力的桌面守护神

ForceBreak 是一款基于 JavaFX 打造的现代化 Windows 桌面健康助手。它能够在系统托盘默默守护你，通过强制全屏锁屏来提醒沉浸在 TradingForex 交易复盘、编写 Telegram 爆仓预警机器人、或是剪辑 AI 短视频的你：**“是时候站起来活动一下，去摸摸狗子了！”**

## ✨ 核心特性

* ⏱️ **动态托盘图标**：告别死板的静态图标。采用 AWT 动态重绘技术，托盘图标实时显示剩余分钟数。最后 60 秒自动切换为**橙色高亮秒数倒计时**，视觉反馈拉满。
* 🖼️ **自定义沉浸式锁屏**：支持自定义锁屏文案与本地配图。你可以放上家人或宠物（比如可爱的柴犬“微笑”）的照片，让休息时间充满温情。
* ⏸️ **人性化防误杀**：倒计时最后 30 秒右下角弹窗预警，支持**一键延时 15 分钟**（底层采用 `AtomicInteger` 保证多线程可见性与原子操作），绝不打断你突然爆发的灵感。
* 🚀 **静默开机自启**：内嵌注册表控制逻辑，支持在软件设置中一键开启/关闭随 Windows 开机启动，无需手动设置快捷方式。
* 📦 **独立免安装 EXE**：抛弃繁琐的环境配置，基于 `jpackage` 将 JDK 运行时与业务代码深度打包。生成的 `.exe` 文件自带独立环境，发给任何 Windows 用户均可**双击秒开**。

## 🛠️ 技术栈与架构演进

* **UI 框架**：JavaFX 21
* **核心语言**：Java 21
* **构建工具**：Maven (`maven-shade-plugin` 打包 Fat Jar)
* **原生打包**：JDK 原生 `jpackage` 工具
* **踩坑记录（GraalVM vs jpackage）**：
  本项目曾尝试使用 GraalVM (`gluonfx-maven-plugin`) 编译 AOT 原生镜像以追求极致体积。但由于 AWT 的 `SystemTray`（系统托盘）模块在 Windows 平台与 GraalVM 的 C++ 底层静态编译存在“生殖隔离”（阉割了 `java.desktop` 的本地调用），最终项目重构，转向使用 `jpackage` + Fat Jar 的方案，完美兼容了现代 JavaFX 界面与底层 AWT 托盘交互，且避免了复杂的环境依赖。

## 🚀 如何运行与构建

### 1. 本地开发调试
要求本地已安装 JDK 21 与 Maven：
```bash
# 编译并运行 JavaFX 程序
mvn clean javafx:run
```

### 2. 一键构建商业级 EXE
项目根目录已提供了一键打包脚本 `build.bat`。在 Windows 环境下：
1. 确保系统环境变量 `Path` 中能找到 `mvn`，且脚本内的 `jpackage` 路径正确指向你的 JDK 21 目录。
2. 双击运行 **`build.bat`**。
3. 终端跑完 4 个步骤后，在 `release\ForceBreak\` 目录下即可获取拥有专属图标的 **`ForceBreak.exe`**。

## ⚙️ 配置说明

软件的所有配置（专注时长、自定义文案、图片路径）会自动持久化保存到用户本地目录中，下次启动无缝读取。
* **开机自启动逻辑**：直接操作 Windows 注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 节点，绿色轻量，不依赖任何第三方 C++ 库。
